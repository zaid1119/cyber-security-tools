# Modular Cybersecurity Toolkit 

A collection of essential security tools developed for network analysis, reconnaissance, and data protection. This project was built to understand the underlying protocols and cryptographic methods used in modern cybersecurity.

# Features

* Network Traffic Sniffer: Real-time TCP/IP packet inspection to monitor data flow.
* VAPT Port Scanner: An automated scanner to identify open ports and potential vulnerabilities on a target host.
* Crypto Vault: A secure module for local data encryption and storage.

# Getting Started

Prerequisites
* Python 3.x
* Administrative/Root privileges (required for raw socket access in the Sniffer)
